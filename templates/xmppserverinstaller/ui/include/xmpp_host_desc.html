To make XMPP work on your server, you need to set up SRV records for domain {{ host }}. It will allow XMPP clients to correctly find the location of your server using Domain Name System (DNS). You need to add several types of DNS records to make things work:

  • A record for subdomain {{ host }} to make server management panel available from the Internet on this address
  • SRV records to help XMPP clients and servers find your server

A Record

Hostname IP Address TTL (seconds)
xabber.{{ host }} Your server IP Default

SRV Record

Service Protocol Priority Weight Port Target TTL (seconds)
_xmpp-client TCP 10 5 5222 xabber.{{ host }} Default
_xmpp-server TCP 10 5 5269 xabber.{{ host }} Default
